# Manny's Construction LLC — Website

Premium one-page website for Manny's Construction LLC, serving Liberal, KS and surrounding communities.

## Quick Start

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
cd frontend
npm run build
npm run preview
```

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS v4
- Framer Motion animations
- Lucide React icons
